PRUEBA 1: Método POST - Crear reporte
URL: http://localhost:3000/reportes
Método: POST
Body JSON enviado:
{
  "tipo": "Infraestructura",
  "descripcion": "Hueco en la vía principal"
}
Respuesta del servidor:
{
  "mensaje": "Reporte registrado",
  "reporte": {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Hueco en la vía principal"
  }
}
Estado: 201 Created ✅
